.

│  ├─ avatar.jpg

## GitHub Pages
1. Crea el repo y sube todo el contenido.
2. Settings → Pages → Deploy from a branch → Branch: `main` / Folder: `/root`.
3. Abre `https://<tu-usuario>.github.io/<repo>/`.
